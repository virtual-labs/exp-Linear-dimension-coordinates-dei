Linear Independence of S ⊆ R2

CASE 1 : S = ϕ

S is LINEARLY INDEPENDENT, by definition.

CASE 2 : (i) S = {a}, a ≠ 0

S is LINEARLY INDEPENDENT

Reason:

α . a = 0 ⇒ α = 0, α ∈ F

(ii) S = {0}, a = 0

S is LINEARLY DEPENDENT

Reason:

α . 0 = 0, α ≠ 0, α ∈ F

CASE 3 : S ≠ {.} , ϕ and 0 ∉ S

S = {a, b}

b ∈ L(S1), where S1 = {a} ⇒ S is LINEARLY DEPENDENT
b ∉ L(S1). where S1 = {a} ⇒ S is LINEARLY INDEPENDENT

Case 4 : 0 ∈ S

0 = α . 0, α ≠ 0, α ∈ F

What do you infer? /

Example 1: S = {(1, 1), (1, 2)}

b ≡ (1, 2) ∉ L({(1, 1)}) ⇒ S is LINEARLY INDEPENDENT
Also, α . (1, 1) + β . (1, 2) = (0, 0)
  ⇒ (α + β , α + 2β) = (0, 0)
  ⇒ α + β = 0 , α + 2β = 0
  ⇒ α = 0 , β = 0

Example 2: S = {(1, 1), (3, 3)}

b ≡ (3, 3) ∈ L({(1, 1)}) ⇒ S is LINEARLY DEPENDENT
Also, (3, 3) = 3(1, 1)
    3(1, 1) + (-1) (3, 3) = (0, 0)
